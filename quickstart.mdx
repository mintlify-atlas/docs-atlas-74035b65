---
title: "Quickstart: your first AI-written LinkedIn post in minutes"
description: "Install Social Media Skills, build your voice profile with voice-builder, and publish your first AI-written LinkedIn post in under 10 minutes."
---

The fastest path to a published post is three steps: install the plugin, run `voice-builder` to create your voice profile, then ask Claude to write a post. This page walks you through each step with the exact commands and phrases to use.

<Info>
  This quickstart uses the Claude Code plugin marketplace (the fastest install method). If you prefer to clone the repo, use a git submodule, or upload individual skills to Claude Desktop, see the [installation guide](/installation).
</Info>

<Steps>
  <Step title="Install the plugin">
    Open a Claude Code session and run the following two commands in order. The first command registers the marketplace source, the second installs all 17 skills into your project.

    ```bash
    /plugin marketplace add charlie947/social-media-skills
    ```

    ```bash
    /plugin install social-media-skills
    ```

    Claude will confirm once the skills are installed and available in your session.
  </Step>

  <Step title="Set API keys (optional, for two skills)">
    Most skills work immediately. If you plan to use `post-scorer` or `reels-scripting`, export the required environment variables before continuing.

    ```bash
    export APIFY_API_TOKEN=your_token
    export GOOGLE_AI_API_KEY=your_key
    ```

    <Note>
      Skip this step if you only want to write posts. `APIFY_API_TOKEN` and `GOOGLE_AI_API_KEY` are only needed for `post-scorer` and `reels-scripting`.
    </Note>
  </Step>

  <Step title="Build your voice profile">
    `voice-builder` is the foundation. Every other skill reads `about-me.md` and `voice.md` before it writes a single word, so run this first.

    In Claude, type:

    ```
    Build my voice
    ```

    Claude will run a short interview asking about your background, audience, and goals, then ask you to paste 3–5 examples of your existing writing. At the end it produces two files in your project:

    - `about-me.md` — your background, audience, and positioning
    - `voice.md` — your writing rules, tone, and style patterns

    <Tip>
      Paste writing samples that represent your best work — posts that performed well, newsletter sections you are proud of, or any copy that sounds most like you. The richer the samples, the more accurate the voice profile.
    </Tip>
  </Step>

  <Step title="Write your first post">
    With your voice profile in place, ask Claude to write a post on any topic:

    ```
    Write me a post about how AI is changing content creation
    ```

    Claude picks the `post-writer` skill, reads your `voice.md` and `about-me.md`, and drafts a LinkedIn post in your voice. It will explain its choices and offer a revision loop.

    Other trigger phrases that route to the right skill automatically:

    | What you type | Skill used |
    |---|---|
    | `"Write me a post about..."` | `post-writer` |
    | `"Score this draft against my history"` | `post-scorer` |
    | `"Make me a carousel from this"` | `gemini-carousel` |
    | `"What should I post this week"` | `niche-research` or `content-matrix` |
    | `"Turn this outlier Reel into a script"` | `reels-scripting` |
    | `"I need a thumbnail for 'How I fired my team'"` | `youtube-thumbnail` |
    | `"Write me a pinned comment"` | `pinned-comment` |
  </Step>

  <Step title="Review and publish">
    Claude outputs the post in a code block and explains why it made each structural choice. You can:

    - Ask for changes: `"Make the hook more direct"` or `"Cut the last line"`
    - Ask for alternatives: `"Give me 3 different hooks for this"`
    - Accept as-is: `"Ship it"` to save the final version to your project

    Copy the final post from the code block and paste directly into LinkedIn.
  </Step>
</Steps>
