## How the system works

Every skill reads your voice profile. Run `voice-builder` once to create `about-me.md` and `voice.md`, then every other skill references those files automatically before writing a single line.

<Steps>
  <Step title="Build your voice">
    Run `voice-builder` to complete a short interview and analyse 3–5 writing samples. Creates `about-me.md` and `voice.md` in your project.
  </Step>
  <Step title="Pick a skill">
    Tell Claude what you want to create. It picks the right skill from trigger phrases like "write a post", "score my draft", or "make me a carousel".
  </Step>
  <Step title="Review and ship">
    Every skill outputs content in a code block, explains its choices, and offers a revision loop. Say "ship it" to save the final version.
  </Step>
</Steps>

## Prerequisites

Two skills require external API keys. All others work out of the box with Claude.

| Variable | Required for |
|---|---|
| `APIFY_API_TOKEN` | `post-scorer`, `reels-scripting` |
| `GOOGLE_AI_API_KEY` | `reels-scripting` (Gemini 2.5 Flash video analysis) |

<Note>
  Image generation skills (`gemini-infographic`, `gemini-carousel`, `quote-post`, `youtube-thumbnail`, `profile-optimizer`) output ready-to-paste prompts for Gemini's Create Image feature — no API key needed.
</Note>
